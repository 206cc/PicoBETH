## **組裝步驟**

以下組裝步驟請搭配 [YouTube 教學影片](https://youtu.be/gtyGDhp-Uqk) 進行。

### **步驟 1：3D 列印部件**
![part1](docs/img_3d_part1.jpg)
- **主體部件 (PART-1)**  
需承受較大的變形力，建議使用以下列印參數：  
  - **壁厚：** 5mm 以上  
  - **填充：** 30% 以上  

![part3](docs/img_3d_part3.jpg)
- **Load Cell 固定架 (PART-3)**  
作為 Load Cell 與滑台座的固定零件，此部件需承受較大的變形力，建議使用以下列印參數：  
  - **填充：** 100%

### **步驟 2：組裝主體 (Part-1)**
1. 使用 **M4 x 30mm 螺絲 + 彈簧墊圈 + 平墊圈** x2，將主體下方與滑台稍微固定（保持可滑動狀態）。
2. 使用 **M4 x 20mm 螺絲 + 彈簧墊圈 + 平墊圈** x4，將主體側面與滑台稍微固定（保持可滑動狀態）。
3. 將主體與滑台推齊。
4. 先將底部 **M4 x 30mm 螺絲** x2 使用 **1.5 Nm** 扭力鎖緊。
5. 再將側面 **M4 x 20mm 螺絲** x4 使用 **1.5 Nm** 扭力鎖緊。

> [!CAUTION]
>  請加裝平墊圈與建議的安裝扭力值進行鎖緊，過大的扭力可能導致 3D 列印部件損壞。

### **步驟 3：組裝 PCB 與 TB6600 安裝架 (Part-4)**
1. 使用 **M3 x 10mm 割尾螺絲** x4 將 TB6600 下方固定。
2. 使用 **M3 x 10mm 割尾螺絲** x2 和 **M3 x 6mm 割尾螺絲** x2 將 PCB 主板固定。
3. 接上 **TB6600 電源線**、**控制訊號線** 與 **馬達訊號線**。
4. 使用 **M3 x 10mm 割尾螺絲** x2 將 TB6600 固定至支架。

### **步驟 4：組裝電源線**
1. 將 **電源開關** 與 **DC 插座** 安裝至 **開關飾蓋 (Part-5)**。
2. 將 **主板電源線** 的接頭插入主板。
3. 使用 **灰色彈簧螺式電線連接器**，將 **DC 插座正極** 與 **電源開關任一線** 接上。
4. 將其餘 **3 條正極線** 接入另一個 **灰色彈簧螺式電線連接器**。
5. 最後使用 **灰色彈簧螺式電線連接器** 接上 **3 條負極線**。

### **步驟 5：組裝排線**
1. 將 **LCD 訊號線 XH2.54mm 4P** 穿過管道，插入主板，並在二端插頭上的 GND 做標記。
2. 依次將 **五向按鍵 XH2.54mm 6P**、**LED 訊號線 XH2.54mm 4P** 和 **取消按鍵 XH2.54mm 2P** 穿過管道，並將接頭插入主板。

### **步驟 6：安裝珠夾頭與 NJ5 20kg Load Cell (YZC-133)**
1. 在 **Load Cell** 上方 **5mm** 處鑽一個直徑 **6mm** 的孔，然後安裝至珠夾頭。
2. 使用 **M5 x 70mm 內六角螺絲 + 平墊圈**，將 **Load Cell** 與 **Part-3 Load Cell 固定架** 鎖上（留 5mm 空隙，暫不鎖緊）。
3. 使用 **M3 x 6mm 圓頭螺絲** 將啟動開關支架固定至珠夾頭。
4. 使用 **M3 x 6mm 割尾螺絲** 固定 **珠夾啟動開關 XH2.54mm 2P**。
5. 整理線路後，使用 **2.0 Nm** 扭力將 **M5 x 70mm 內六角螺絲** 鎖緊。
6. 使用 **M4 x 16mm 螺絲 + 彈簧墊圈 + 平墊圈** x4，並以 **1.5 Nm** 扭力鎖緊。
7. 將 **珠夾啟動開關 XH2.54mm 2P** 的接頭插入主板。
8. 使用 **捲繞管 Ø4** 整理 YZC-133 與珠夾啟動開關的線路。

> [!CAUTION]
>  請加裝平墊圈與建議的安裝扭力值進行鎖緊，過大的扭力可能導致 3D 列印部件損壞。

### **步驟 7：安裝前限位開關與後限位架 (Part-2)**
1. 使用 **M2.6 x 10mm 割尾螺絲** x2 將 **後限位開關 XH2.54mm 2P** 固定於 **後限位架 (Part-2)**。
2. 將 **後限位架 (Part-2)** 安裝至 **主體 (Part-1)**，並使用 **M3 x 6mm 割尾螺絲** 鎖緊。
3. 使用 **M2.6 x 10mm 割尾螺絲** x2 將 **前限位開關 XH2.54mm 2P** 固定於 **主體 (Part-1)**。
4. 將前、後限位開關的接頭插入主板。

### **步驟 8：安裝電源與開關飾蓋 (Part-5)**
1. 將 **電源與開關飾蓋 (Part-5)** 安裝至 **主體 (Part-1)**，並使用 **M3 x 6mm 割尾螺絲** x2 鎖緊。
2. 整理電源線路。

### **步驟 9：組裝 LCD 與按鍵架 (Part-6)**
1. 將 **LCD 與按鍵架 (Part-6)** 定位至 **主體 (Part-1)** (先不固定)。
2. 將 **LCD 訊號線 XH2.54mm 4P** 接至 **2004 i2c LCD**（注意 GND 標記），並使用 **M3 x 6mm 割尾螺絲** x4 固定。
3. 依次將 **五向按鍵 XH2.54mm 6P**、**LED 訊號線 XH2.54mm 4P** 和 **取消按鍵 XH2.54mm 2P** 的接頭插入 **PCB 按鍵板**。
4. 使用 **M3 x 6mm 割尾螺絲** x4 將 **PCB 按鍵板** 固定至 **LCD 與按鍵架 (Part-6)**。
5. 使用 **M4 x 8mm 內六角螺絲** 將 **LCD 與按鍵架 (Part-6)** 固定至 **主體 (Part-1)**。

### **步驟 10：安裝穿線機頭至平台**
1. 安裝前需設計一個底部轉接座，以適配自已的穿線機平台。可參考 [EP.9 定位與固定](https://youtube.com/Ax4agdsqyms)。
2. 建議使用 M8 規格螺絲，並搭配平墊片，遵循建議的扭力值 **3.0 Nm**。

### **步驟 11：硬體測試**
1. 開機時同時按住 **中鍵** 進入 HW Testing Mode。
2. 根據提示完成所有硬體測試。

### **步驟 12：張力校正與測試**
1. 參考 [EP.10 最後設定 (張力校正)](https://youtube.com/WMFJQU_TSFk) 完成張力校正。
2. 測試 20LB 和 30LB 張力是否準確。
3. 校正完成後，記得至選單內按下 **"S"** 儲存校正參數。

## 其它注意事項

### Load Cell 感應線

![hx711_loadcell](docs/img_hx711_loadcell.jpg)

由於設計佈局的限制，Load Cell 的安裝方向與 HW1 和 HW2 相反。因此，接線時需對調接頭上的綠色與白色訊號線，才能正確偵測受力方向。若未正確接線，可能導致張力偵測方向錯誤。

### Load Cell 安裝方向

請注意 Load Cell 的安裝方向。影片中展示的是 NJ5 出廠的 YZC-133，其安裝方向與大多數 YZC-133 不同。若使用其他廠牌的 YZC-133，安裝方向可能需要旋轉 180 度。詳細說明請參考 [EP.5 補充篇 - 斜度微調與 YZC-133 Load Cell](https://youtube.com/22Ev_kWTnxk)。
